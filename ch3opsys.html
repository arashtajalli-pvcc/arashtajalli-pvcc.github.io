Operating Systems Terminology
Term Definition

Operating System

 Operating System is a fully integrated set of specialized programs that handle all the operations of the computer.

Job Queue

An ordered list of jobs waiting to be processed by a subsystem

FIFO

abbr. First In First Out; a method for organizing the manipulation of a data structure where the oldest entry is processed first.

Interactive Processing

The act of interacting directly with a program and/or data for immediately modifying, retrieving, and/or displaying information.

Time-sharing

multiple users with different programs interact simultaneously with the central processing unit of a large scale computer

Multiprogramming

a form of parallel processing in which several programs run at the same time on a uniprocessor system.

Application Software

A computer program designed to carry out a specific task other than the one relating to the computer itself.

System Software

A program designed to run a computer's hardware and applications and manage its resources.

Utility Software

A program specifically designed to help manage and tune system or application software.

Graphical User Interface (GUI)

a form of User Interface that allows users to interact with electronic devices through graphical icons and audio indicators, instead of text based UI's.

Shell

A computer program that exposes an operating system's services to a human user or other programs.

Kernel

A computer program that is the central manager of hardware resources and what processes needs them.

Device Driver

a group of files that enable one or more hardware devices to communicate with the computer's operating system.

File Manager

A computer program that provides a user interface to manage files and folders.

Memory Management

A form of resource management applied to computer memory to dynamically allocate portions of memory to programs at their request.

Virtual Memory

A memory management technique that provides an idealized abstraction of the storage sources that are actually available on a given machine.

Boot Strapping

To boot or lad a program into a computer using a much smaller initial program to load in the desired program.

ROM

A type of computer storage containing non-volatile permanent data that, normally, can only be read not written to.

Deadlock

A situation in which 2 computer programs sharing the same resource are effectively preventing each other from accessing the resource.

Spooling

A process in which data is temporarily held to be used and executed by a device, program, or the system. (i.e. RAM usage)

Super User (or Administrator)

A special user account used for system administration and has unrestricted access to all files and commands aka root.

multi-factor authentication

An electronic authentication method in which a user is granted access to a website or application only after successfully presenting 2 or more pieces of evidence to an authentication mechanism.
