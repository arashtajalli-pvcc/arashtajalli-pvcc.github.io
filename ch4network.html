Network Classifications
  • LAN - Local Area Network
  • MAN - Metropolitan Area Network
  • WAN - Wide Area Network

    Network Models
  • bus network - A LAN topoglogy in which each node (a workstation or device) is connected to a main cable or link called a bus.
  • star network - A LAN topology in which all nodes are directly connected to a common central computer (i.e. the hub) rather than a bus connection.

    • Protocols
  • CSMA/CD Ethernet - Carrier Sense Multiple Access/Collision Detection, a media access control method used in early ethernet/LANS when there was a shared Bus Topology and each computer was connected by coaxial cables.
  • CSMA/CA wifi - Carrier Sense Multiple Access/Collision Avoidance - a protocol for carrier transmission in 802.11 networks, developed to minimize the potential of a collision occurring when two or more stations send their signals or a data link layer.

    • Combining networks:
  • repeater - a network device that amplifies or regenerates an incoming signal before retransmitting it.
  • bridge - Used to connect 2 subnetworks that use interchangable Protocols, i.e. combine 2 LANs to form an extended LAN.
  • switch - Networking device as Layer 2 (Data link) which connect devices in a network and use packet switching to send, receive, or forward data packetr or data frames over the network.
  • router - Network device at layer 3 (Network Layer) which is responsible for receiving, analysing, and forwarding data packets amongst the connected computer networks; data packet arrives, router inspects destination address, consults the routing tables to decide the optimal route and then sends the packet on that route.
  • gateway - a network node that forms a passage between 2 networks operating with different transmission protocols. Acts as the entry-exit point for a network.

    • Client/server model
  • P2P: peer-to-peer - a network in which each participant (peer) can act as both client and a server, allowing to share resources and information directly with one another without the need for a central server, aka decentralized.
  • distributed system - A collection of autonomous computer systems that are physically separated but are connected by a centralized computer network that is equipped with distrubuted system software. the computers will share resources, and files assigned to them.
  • cloud computing - on demand availability of computer system resources, esp. data storage and computing power without direct active management by the user.

    IP address
  • ICANN

    • Internet Corporation for Assigned Names and Numbers - a group and nonprofit organization responsible for the maintenance and procedures of several databases related to the namespaces and numerical spaces on the internet.

  • DNS - A hierarchical and distributed naming system for computers, services, and other resources in the internet or other Internet Protocol networks. (i.e. learn.vccs.edu)
  • TLD: Top-level domain - the last segment of a domain name. (i.e. ".com",".org", etc.)
  • Country-code TLDs

    • An internet top-level domain reserved for a country, sovereign state, or dependent territory.

    Internet Protocols
  • FTP - File Transfer Protocol - a communication protocol used for the transfer of computer files from a server to a client on a a computer network.
  • HTTP - Hyper Text Transfer Protocol - a protocol used for exchanging information over the internet.
  • SMTP - Simple Mail Transfer Protocol - An internet standard communication protocol for electronic mail transmission.

    • World Wide Web
  • URL - Uniform Resource Locater - a reference to a resource that specifies its location on a computer network and a mechanism for retrieving it.
  • client-side activities - Everything in a web Application that is displayed or takes place on the side of the client.
  • server-side activities - Everything that happens on the server i.e. rendering dynamic webpages, interacting with databases, identity authentication, and push notifications.

    • Internet four-layer software
  • Application layer - Human-computer interaction layer - where applications can access the network services.
  • transport layer - Transmits data using transmission protocols including TCP and UDP.
  • network layer - Decides which physical path the data will take.
  • link layer - transfers data between nodes on a network segment across the physical layer.

    • Internet is:
  • packet-switching networt: the transfer of smmall pieces of data across various networks.
  • fault tolerant - a process that enables an operating system to respond to a failure in hardware or software.
  • redundant - a system design in which a component is duplicated so if it fails there will be a backup.

    Cybersecurity
  • Malware - software specifically designed to disrupt, damage, or gain unauthorized access to a computer system.
  • Virus - a type of malware that, when executed replicates itself by modifying other computer programs and inserting its own code into those programs.
  • Worm - a type of malware whose primary function is to self-replicate and infect other computers while remaining active on infected systems.
  • Trojan Horse - a malware that misleads users of its true intent by disguising itself as a standard program.
  • spyware - software with malicious behaviour that aims to gather information about a person or organization and send it to another entity.
  • phishing - the fraudulent practice of sending emails or other messages purporting to be from reputable companies in order to induce individuals to reveal personal information, such as passwords and credit card numbers.
  • DDoS attack - Distributed Denial of Service attack - a cyber-attack in which the perpetrator seeks to make a machine or network resrouce unavailable to its intended users by temporarily or indefinitely disrupting services of a host connected to a network.

    • Protection
  • firewall - a network security system that monitors and controls incoming and outgoing network traffic based on predetermined security rules.
  • Spam filter - a program used to detect unsolicited, unwanted and virus-infected emails from getting into a user's inbox.
  • proxy server - a server application that acts as an intermediary between a client requesting a resource and the server providing that resource.
  • antivirus software - anti-malware, a computer program used to prevent, detect, and remove malware.
  • cryptography - The process of hiding or coding information so that only the prson a message was intended for can read it.
  • public key encryption - asymmetric cryptography - is the field of systems that use pairs of related keys. a public and private key.
